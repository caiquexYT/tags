Quando um jogador tem múltiplas badges, elas são ordenadas da maior para menor prioridade da esquerda para a direita.

Regras de Exibição

SituaçãoBadge exibida
Jogador presente no badges.json ✔ Badges definidas no JSON
Jogador usa o client mas não está no JSON ⚡ Badge usuario (automática)
Jogador não usa o client ✖ Nenhuma badge

Formato do JSON

{
  "players": [
    {
      "nick":   "DevOwner",
      "badges": ["dev"]
    },
    {
      "nick":   "SuportePremium",
      "badges": ["suporte", "premium"]
    },
    {
      "nick":   "CanalYT",
      "badges": ["youtuber"]
    }
  ]
}

O campo nick é case-insensitive. Múltiplas badges por jogador são suportadas.
